## What is an HLR Lookup?

An **HLR lookup** (Home Location Register lookup) is a real-time query sent directly to mobile network operators to determine the exact status of a mobile phone number.

Platforms like [1Lookup](https://www.1lookup.io/) use this technology to tell you if a number is currently active, turned on, roaming, or ported to a different network. By pinging the carrier silently, an HLR lookup allows businesses to clean their contact databases, ensuring they only send SMS messages or make calls to valid, reachable users. This dramatically reduces telecom costs and improves campaign ROI.
